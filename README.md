My keyborad shortcuts for cocntroling media player.

## Install AutoHotKey
[AutoHotKey](https://www.autohotkey.com/)

## Volume up
```[win]+[WheelUp]```

## Volume down
```[win]+[WheelDown]```

## Next song
```[win]+[mouse_right]```

## Previous song
```[win]+[mouse_left]```

## Pulse song
```[win]+[mouse_middle]```
